### 연산자
- 산술 연산자: + - * /(나누기 값) //(나누기 몫) %(나누기 나머지)
- 지수 연산자: ** (제곱)
- 대입(복합) 연산자: = += -= *= /= //= %=
- 관계(비교) 연산자: > >= < <= == !=
- 논리 연산자: not and or
- 삼항 연산자: A if 조건식 else B 


```python
num1 = 7
num2 = 2

print(num1/num2) #나누기
print(num1%num2) #나머지
print(num1//num2)#몫
```

    3.5
    1
    3
